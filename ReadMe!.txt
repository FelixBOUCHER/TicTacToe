Elements employés :
- Un jeu de Tic-Tac-Toe open-source, sous Unity, programmé par Thomas Knox.
- Template de xtralife pour Unity trouvé sur le site GitHub.

Fonctionnalités mises en place dans le cadre du test technique :

-> Intégration du jeu Unity sur la plate-forme développeur xtralife.

-> Mise en place d'un système de login
- - -> Bouton pour se login anonymement.
- - -> Bouton pour transformer son compte anonyme en compte email.
- - -> Bouton pour se login via email.
- - -> Logins possibles depuis le menu.

-> Mise en place d'un système de leaderboard
- - -> Scores automatiquement enregistrés sur trois leaderboards séparés selon le niveau de difficulté de l'IA affrontée (Easy, Normal, Hard).
- - -> Les scores sont calculés selon le nombre de victoires obtenues dans une session de parties consécutives.
- - -> Leaderboards consultables depuis le menu.

-> Mise en place d'un système d'achievements
- - -> Achievements d'un compte consultables depuis le menu.
- - -> Achievement 3XPower : Battre l'IA Hard trois fois dans une même session (l'IA Hard reset le compteur de victoires à 0 à chaque défaite).
- - -> Achievement Failure! : Perdre contre l'IA Easy.

-> Amélioration de l'interface par rapport au projet open-source utilisé.

NOTE IMPORTANTE : Pour lancer le jeu depuis l'interface Unity et tester les fonctionnalités implémentées, il est nécessaire de lancer la Scène "Menu" trouvable dans Assets/Scenes.
